1サマリー

SNS運用事業とWeb制作事業の2〜3年間の売上見通し・必要人員・投資計画をまとめたドラフト。

基本方針

  • Web制作で入り込む → MEO → SNS運用 の段階的アップセル
  • SNS事業は2〜3年かけて正しく成長させる（無理に急がない）
  • 整骨院4,000院・整備工場ネットワークを最大限活用
  • 夜職で培ったSNSノウハウをパーソナルジム → 高単価飲食 → 美容クリニックへ横展開
  • 最終目標：来店型ビジネスの「外部マーケティング部」のポジション確立

2事業概要

SNS運用事業

クライアント企業のSNS戦略立案・運用代行・インハウス化支援事業。
メインSNS: TikTok中心。Instagram・YouTubeも対応。
現在のメインターゲット: 夜職（バー/コンカフェ）。今後パーソナルジム・高単価飲食・美容クリニックへ横展開予定。

SNS事業の差別化ポイント

  • 「人が主役」の動画設計: スタッフ・トレーナー・キャストの個性で認知を取るノウハウ
  • 来店導線の一気通貫: SNS → DM/LINE → 予約 → 来店 → リピートまで設計
  • 撮影〜編集のチーム体制: 栗原さんの映像力 + 上田のディレクション
  • 月額プラン体系が確立済み: 15万〜50万、段階的なアップセル設計
  • インハウス化支援: 最終的にクライアント自身でSNS運用ができる状態まで伴走。自走後はサポート/保守（3万/月）で継続関係を維持

ターゲット設計（夜職）

区分内容
対象業態バー・コンカフェ（両方対応）
想定顧客層社会人
目標新規獲得に加え「新規の常連化（リピート・固定客化）」を創出
対応エリア全国対応（撮影は当面関東のみ）
ゴール永続契約ではなく、インハウス化（クライアント自走）を目指す

最低要件:

  • キャスト露出（顔出し）での発信に協力可能
  • 店長/オーナーが意思決定でき、運用ルールを守れる
  • 予約/来店導線（DM/LINE等）を整える意思がある

除外条件:

  • 店内ルールが曖昧で炎上・規約リスクが高い
  • キャスト協力が得られず素材供給が止まる
  • 「バズだけ」要求が強く、来店/常連化の合意がない

ポジショニング

「キャストを主役にしたSNSで認知を取り、来店導線を整え、集客・採用の効率化までコミットする」

  • 認知獲得（ショート動画中心）
  • プロフィール/固定投稿/ハイライトでの検討促進
  • DM一次対応と予約導線整備（回収）
  • 常連化の仕組み（再来理由の設計）

Web制作事業

自動車整備工場・中古車販売会社・整骨院に特化した、低価格・高品質のWebサイト制作サービス。
テンプレート × AI活用で、1人でも月5〜8件の制作が可能。
戦略的位置づけ: SNS運用事業への「入口商品」。Web→MEO→SNSと段階的にアップセルする導線。

Web制作事業の強み

  • 事故なびネットワーク: 4,000院の整骨院 + 整備工場パートナーに直接営業可能（リスト不要）
  • 圧倒的な低価格: 一般Web制作会社の1/10以下（5万円〜）。テンプレート+AIで実現
  • Web × SNS × 動画のワンストップ: 他社はWeb「だけ」。Opus.netは全部まとめて対応
  • 保守収入のストック効果: 月額1万円×件数が積み上がり、安定収入基盤に

アップセル構造

Web制作（5〜10万）→ MEO（+5,000/月）→ SNS体験パック（1万）→ SNS運用（15〜50万/月）

Web制作で接点を作り、成果を見せながら段階的に高単価サービスへ移行する設計。

業界横展開の戦略（売上シミュレーションの根拠）

夜職（バー/コンカフェ）で培った「人を主役にした動画設計」「来店導線の構築」「常連化の仕組み」を他業界に転用する。
業界を広げることで、SNS事業の成長速度と安定性を同時に高める。

コア能力と転用先

夜職で培ったコア能力転用先の業界転用の考え方
人＝コンテンツの動画設計パーソナルジム・美容クリニックトレーナーや施術者が主役。夜職の「キャスト紹介」と構造が同じ
来店導線（SNS→DM/LINE→予約）全業界共通来店型ビジネスはすべて同じ導線設計が使える
常連化・リピート設計高単価飲食・ジムリピーター獲得の仕組みがそのまま応用可能
撮影〜編集のチーム体制全業界共通栗原さんの映像力は業界問わず強み

13リスクと対策

リスク影響対策
上田依存が解消できない 成長が月商150万あたりで頭打ちになる Q3のSNSディレクター採用が最重要。採用が遅れる場合はWeb制作を優先しSNSの新規獲得を抑える
横展開が想定より遅れる SNS売上がFY2026で870万（保守的）に留まる 夜職のクライアントを着実に増やしつつ、Web制作のストック収入でカバー
Web制作の需要が想定以下 保守収入の積み上げが遅れる 事故なびメルマガで4,000院に一斉案内。反応が薄ければ価格・プラン見直し
解約率が想定以上 MRRの成長が鈍化 月次レポートの充実、定期的な価値提供で解約を防ぐ。解約理由を分析しサービス改善
採用したディレクターが合わない コストだけかかり品質が下がる 最初3ヶ月は業務委託でお試し。合わなければ切り替え可能な契約にする
